
Flask-AppUtils
==============
A collections of common utilities and patterns for Flask applications with useful snippets from [flask.pocco.org](flask.pocco.org)


Install
-------

```sh
$ pip install flask-apputils
```
    

Modules
-------

* `decorators`
* `filters`
* `helpers`
* `routing`
* `middlewares`
* `mimes`
* `templating`
* `validators`
